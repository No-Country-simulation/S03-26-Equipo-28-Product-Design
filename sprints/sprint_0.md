# 🎨 Sprint 0 

## 🏢 Sector
EdTech

## 📅 Duración
Semana 0

---

## 🎯 Objetivo del Sprint

Comprender el problema, analizar la necesidad del cliente y sentar las bases del sistema de diseño multibrand.

---

## 🧩 Contexto del Proyecto

Empresas con múltiples productos digitales enfrentan inconsistencias visuales y de experiencia.  
El objetivo es crear un Design System centralizado, escalable y adaptable a múltiples marcas dentro del sector EdTech.

---

## 📌 Alcance del Sprint 0

## 👥 Formación del Equipo

El equipo está conformado por 3 integrantes con enfoque en Producto y UX/UI:

- Product Designer → encargado de estrategia, definición del problema y alineación con objetivos del negocio.
- UX Researcher → responsable de investigación de usuarios y análisis de insights.
- UI Designer → encargado de la parte visual, componentes y consistencia gráfica.

---

## 📌 Roles y responsabilidades

| Rol | Responsabilidades |
|-----|------------------|
| Product Designer | Definición de objetivos, alcance del sistema, toma de decisiones |
| UX Researcher | Benchmark, investigación, user insights |
| UI Designer | Diseño visual, componentes, UI Kit |
