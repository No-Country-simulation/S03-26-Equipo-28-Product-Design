🎨 Multi-Brand Design System
Sistema de diseño escalable para productos EdTech con múltiples marcas.

🚀 Objetivo
Centralizar la identidad visual, componentes y patrones de interacción para garantizar consistencia entre productos digitales, permitiendo al mismo tiempo la personalización por marca.

🧩 Arquitectura del Sistema
El sistema se organiza en 4 niveles:
Foundations → Tokens base (color, tipografía, spacing)
Components → UI reutilizable
Patterns → Flujos y comportamientos
Themes → Variantes por marca

🎯 Principios
Consistencia sobre personalización
Accesibilidad desde el diseño
Escalabilidad modular
Documentación como producto

🎨 Estrategia Multibrand
El sistema se basa en:
Core (70%) → Compartido entre todas las marcas
Brand Layer (30%) → Personalización controlada

📦 Design Tokens
Core tokens → valores base
Semantic tokens → significado contextual
Theme tokens → overrides por marca
