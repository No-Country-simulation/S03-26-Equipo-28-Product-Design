---

## 🚀 Objetivo
Centralizar la identidad visual, componentes y patrones de interacción 
para garantizar consistencia entre productos digitales, permitiendo 
al mismo tiempo la personalización por marca dentro del ecosistema EdTech.

## 🎯 Principios de Diseño

- **Consistencia sobre personalización** — el 70% del sistema es compartido
- **Accesibilidad desde el diseño** — WCAG 2.1 AA verificado en las 4 marcas
- **Escalabilidad modular** — nuevas marcas sin reconstruir la base
- **Documentación como producto** — cada componente documentado con uso, estados y tokens
